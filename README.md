# User-management-system-Gestor-de-Utilizadores

User Management System

A simple user management system built with Python.

This project allows the creation, listing, updating, and deletion of users. It was developed as a practice project for backend logic, data handling, and clean code structure.

---

Features

- Create new users
- View all registered users
- Update user information
- Delete users
- Store user data locally

---

Technologies Used

- Python
- JSON data storage
